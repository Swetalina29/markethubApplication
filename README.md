🛒 MarketHub
A full-stack e-commerce marketplace application where users can browse, search, filter, and purchase products across multiple categories.
🌐 Live Demo

Frontend: https://Swetalina29.github.io/markethubApplication
Backend API: https://markethubapplication-production-c91e.up.railway.app/api/products

🛠️ Tech Stack
Frontend

React.js (v18)
Axios for API calls
CSS3 with responsive design
Deployed on GitHub Pages

Backend

Java Spring Boot (v3.2.5)
Spring Data JPA
PostgreSQL database
REST API
Deployed on Railway

✨ Features

Browse 30+ products across 6 categories: Electronics, Clothing, Books, Home, Sports, Other
Search products by keyword
Filter products by category
Add products to cart
Sell/add new products
Responsive design for mobile and desktop
Real-time stock tracking
